# 🌍 Lorde Turismo - Site Institucional

## 📋 Descrição
Site moderno e profissional para a agência de viagens Lorde Turismo, especializada em viagens internacionais sob medida.

## ✨ Características

### Design
- Interface moderna com animações suaves
- Paleta de cores da marca (preto + azul)
- Totalmente responsivo (mobile, tablet, desktop)
- Efeitos visuais avançados (parallax, hover 3D, partículas)

### Seções
1. **Hero** - Chamada principal com estatísticas
2. **Destinos** - Showcase de destinos internacionais
3. **Serviços** - Detalhamento dos serviços oferecidos
4. **Experiências** - Diferenciais da agência
5. **Sobre** - Apresentação da empresa
6. **Depoimentos** - Avaliações de clientes
7. **Contato** - Formulário integrado com WhatsApp

### Funcionalidades
- ⚡ Navegação suave entre seções
- 📊 Contadores animados
- 💬 Integração com WhatsApp
- 📱 Menu mobile responsivo
- 🎨 Animações ao scroll
- 🔝 Botão voltar ao topo
- 📈 Barra de progresso
- 🖼️ Lazy loading de imagens

## 🚀 Como Usar

### Visualização Local
1. Baixe os arquivos (index.html, styles.css, script.js)
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Pronto! O site está funcionando

### Hospedagem
Você pode hospedar em qualquer serviço:
- **Netlify** (gratuito) - Recomendado
- **Vercel** (gratuito)
- **GitHub Pages** (gratuito)
- Qualquer servidor de hospedagem web

#### Deploy no Netlify (mais fácil):
1. Acesse [netlify.com](https://netlify.com)
2. Arraste a pasta do projeto
3. Pronto! Seu site está no ar

## 📝 Personalização

### Alterar Cores
Edite as variáveis CSS no início do arquivo `styles.css`:
```css
:root {
    --primary-color: #1e90ff;
    --secondary-color: #00bfff;
    /* ... outras variáveis */
}
```

### Alterar Conteúdo
Edite o arquivo `index.html` diretamente - o código está bem comentado e organizado.

### Adicionar Novos Destinos
No arquivo `index.html`, copie um card de destino existente e altere as informações.
